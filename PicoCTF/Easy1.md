The way to solve it is by using this table to map values. Our first value of the key is *S* so we look at the y axis column for *S* We then look for the letter *U* in it which is 3 letters to the right of y axis.
After finding it we can then match this to the x axis value. which is *C*.

Doing this for all the values we get the flag as CRYPTOISFUN.

This is our Flag.
